# 3D-Printing---Pinball
A small game to kill time, it uses a ball, 2 bumpers, 3 springs, a case, some fans and a layout for the ball to move around

# How to build it
 First you print the base of the pinball found in the Baza Pinball G-Code, then you build the components, for the bumpers you need to glue them together to work. For the bumpers you must first insert the mechanism before placing the button, and you lock it in by rotating the button itself. To insert the puncher there is a hole in the ball releasing tunnel, which is done so that you can launch the ball by pulling it down from the top. The rest of the components have designated gaps to insert the components, have fun.
